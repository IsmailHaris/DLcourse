Goal of the practical session

The goal of this practical session is threefold:
  1. to learn how to use pre-trained object-detection Convolutional Networks (Yolo, Faster_RCNN, etc);
  2. to learn how to train a Convolutional visual object detectors for custom object categories.

1/ Perform visual objects detection with pre-trained specific ConvNets

YOLO_v3 is currently among the fastest Convolutional visual object detectors. Download the yolo-v3_execute-pretrained.ipynb notebook, and Yolo_v3.rar large (1,5 Go) archive containing all necessary image data, and use it to experiment the main detection parameters (detection threshold, etc...) on real and synthetic video sequences of a front camera placed behind the windshield of a car



2/ Train a Faster_RCNN (or other type of ConvNet) visual objects detector


One of the most comprehensive and up-to-date library for Convolutional visual objects detectors is the Open MMLab Detection Toolbox. It contains implementations of ALL flavors of R-CNN (Fast_R-CNN, Faster_R-CNN, Mask_R-CNN, Cascade_R-CNN), as well as SSD and RetinaNet, with either ResNet or VGG backbone. For several (or all?) of these architectures, Open_MMLab Detection Toolbox includes models pre-trained on VOC dataset or COCO dataset, and code to retrain or fine-tune them.

Two good tutorial+demo notebooks are provided in https://github.com/Tony607/mmdetection_object_detection_demo. Starting from one of these example notebooks (which probably need some adaptation for running locally on your computer):
  •  experiment a pre-trained_on-coco Faster_RCNN (or other architecture) ON THE SYNTHETIC IMAGES FROM CARLA SIMULATOR (provided in the big rar file);
  • retrain it after adding some examples of pedestrians and traffic lights to be created by adding annotations to some of the CARLA image
